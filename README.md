# Getting Started?

The [rcsfa_github_best_practices.md](https://github.com/river-corridors-sfa/RCSFA_GitHub_Organization_Guidance/blob/31a6329151abc389179db1d431d1a645dc1ca4f1/rcsfa_github_best_practices.md) document offers guidance on setting up and managing repositories (also referred to as "repo") within the [River Corridor SFA GitHub Organization](https://github.com/river-corridors-sfa). It covers the following concepts: 
- How to set up a repo within the `river-corridor-sfa` GitHub Organization
	- How to name your repo
	- How to add topics to your repo  
- How to organize your repo
- Expectations for collaboration
